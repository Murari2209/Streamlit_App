## File: app_basic.py

A simple app where users enter their name and receive a greeting message.

Features

Text input
Button interaction
Session state usage
Dynamic output

## File: app_dashboard.py

A mini dashboard displaying monthly sales data.

Features

Month selection dropdown
Metric display
Data visualization
Interactive dashboard

Dashboard Example
Month	Sales
January	1200
February	1500
March	900
April	2000

## File: app_product_form.py

A simple product entry form allowing users to input product details.

Features

Sidebar input
Dropdown category selection
Number input
Button-triggered actions
Display of submitted data

Inputs
Product Name
Category
Price

## File: price_calculator.py

This app calculates the final price after applying a discount.

Features

Product price input
Discount slider
Automatic discount calculation
Table result display

Formula Used
Final Price = Original Price - (Original Price × Discount / 100)


## Run any Streamlit file using:
